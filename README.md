# NIST-Based Incident Response Policy

## Project Overview

This project contains a cybersecurity policy writing sample based on NIST SP 800-61 Rev. 3, *Incident Response Recommendations and Considerations for Cybersecurity Risk Management*. The purpose of this project is to demonstrate security policy writing, incident response planning, documentation, and cybersecurity governance concepts.

This document is an academic portfolio project and is not an official organizational policy.

## Skills Demonstrated

* Security policy development
* Incident response planning
* NIST guidance interpretation
* Governance, risk, and compliance documentation
* Technical writing and professional documentation
* Policy structure, revision tracking, and references

## Policy Focus

The policy focuses on incident response requirements, including:

* Authority
* Purpose
* Scope
* Roles and responsibilities
* Policy statements
* Definitions
* Revision history
* References

## Source Framework

This policy is based on:

National Institute of Standards and Technology. (2025). *Incident Response Recommendations and Considerations for Cybersecurity Risk Management: A CSF 2.0 Community Profile* (NIST Special Publication 800-61 Revision 3). U.S. Department of Commerce. https://doi.org/10.6028/NIST.SP.800-61r3
